感谢您对 Android 的关注！您可以通过以下几种方式进入 Android 的世界并帮助我们改进 Android。如需了解 Android 项目的背景和我们的目标，请查看概览页面。

报告错误

您可以帮助我们改进 Android 的最简单和最有效的方式之一是提交错误。如需了解详情，请访问报告错误页面。

请注意，我们无法保证会在任何特定版本中修复任何特定错误。要了解在您报告错误之后会出现什么情况，请阅读错误的生命周期。

开发应用

我们构建 Android 的一个目标就是方便所有开发者通过一个开放的平台向用户分发应用。您可以帮助 Android 的最佳方式之一就是编写用户喜欢的超酷应用！

要开始开发应用，请访问 developer.android.com。该网站为您提供了使用 SDK 编写适用于 Android 兼容设备的应用所需的信息和工具。

贡献代码

代码是一切的基础。我们很乐意审核您提交的任何更改，因此请检查我们的源代码，找出错误或功能方面的可改进之处，然后开始编码。请注意，您提交的补丁程序越小、越有针对性，就越方便我们进行审核。

要开始帮助改进 Android，您可以通过左侧的链接了解补丁程序的生命周期、git、repo 以及其他工具。您还可以在我们的 Gerrit 服务器上查看所有贡献的动态。如果您在此过程中需要帮助，可以加入我们的论坛。
